## About Me

I am **JoJo**, a Computer Science student at **The Hong Kong Polytechnic University**. I like turning rough ideas into working software quickly, then shaping them into maintainable products with clear architecture, reliable data flow, and deployable systems.

My current focus is **full-stack web development**, **system design**, **architecture design**, and **AI-assisted rapid prototyping / vibe coding**. I am also interested in **agentic workflow design**, including Dify-style orchestration for AI applications.

```txt
idea -> prototype -> architecture -> product
```
